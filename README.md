# 📊 SupportDesk Analytics

An end-to-end Support Analytics project built using **MySQL** and
**Power BI** to simulate a real-world enterprise support environment.

## Project Overview

SupportDesk Analytics is a fictional enterprise support platform
designed to analyze customer support operations and provide actionable
insights for management.

### Workflow

-   Database Design
-   Sample Data Generation
-   SQL Analytics
-   Power BI Data Modeling
-   DAX Measures
-   Interactive Dashboards

## Business Problem

This project answers questions such as: - Which products generate the
highest number of tickets? - Which engineers handle the most workload? -
Which customers require the most support? - Are SLAs being met? - Which
modules require engineering attention?
