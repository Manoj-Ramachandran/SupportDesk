# 🚀 Project INSIGHT – Enterprise Support Analytics Platform

## Overview
Project INSIGHT is an end-to-end Enterprise Support Analytics Platform for analyzing enterprise customer support operations.

## Business Problem
- Which products generate the most tickets?
- Which modules are unstable?
- Are SLA targets met?
- Which engineers are overloaded?
- Which customers require attention?

## Technology Stack
- PostgreSQL
- SQL
- Python
- Power BI
- GitHub

## Dashboard Pages
1. Executive Overview
2. Customer Analytics
3. Engineer Analytics
4. Product & Release Analytics

## KPIs
- Total Tickets
- Open Tickets
- Closed Tickets
- SLA Compliance
- Average Resolution Time
- CSAT
- Engineer Productivity
- Product Stability Index
